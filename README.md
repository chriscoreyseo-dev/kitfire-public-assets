# kitfire-public-assets

Public CDN for KitFire / TecHive / MCG / MarketHive marketing assets.

Any file under this repo is reachable at:
`https://raw.githubusercontent.com/chriscoreyseo-dev/kitfire-public-assets/main/<path>`

## Layout

```
main_newsfeed/    MarketHive main newsfeed hero images, videos, digital
techive/          TecHive Automations group + brand assets
kitfire/          KitFire.ai brand + product visuals
mcg/              Marketing Center Group assets
    └── images/  videos/  digital/
```

## Naming

`YYYY-MM-DD_short-slug.ext` — e.g. `2026-07-05_ferrari-boot.png`

## What lives elsewhere

- **Prompts** for generated assets → private `kitfire-ai-os` repo (strategic).
- **State of play, session logs, decisions** → private `kitfire-ai-os` repo.
- This repo is intentionally **image/video/document only** — finished creative you don't mind being publicly readable.
